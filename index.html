Browser tools

Browser tools have come a long way in the past few years, and they're pretty powerful tools now. Using the native tools built into your browser, you can test how your site performs on differently-sized screens, make quick changes to code and test them live, and debug issues more quickly.

Activity 1

Use your browser's built-in tools to change the colours on this page.

  1. Change all the links to pink.
  2. Change the link hover colour to blue.
  3. Change this link only to bold text.

Activity 2

Use the console to interact with the page using Javascript.

  1. Open the console to find the secret message hidden on this page.
  2. Enter a command in the console to create a browser alert.
